Mecanizado en CNC

Control Numérico Computarizado

Entrada la década del ’50 se introdujo en Estados Unidos el concepto de control numérico (CN) en una fresadora, que usaba tecnología de válvulas de vacío y la carga de datos se realizaba mediante tarjetas perforadas. Ya en los años ’60 la válvulas de vacío eran reemplazadas por transistores, hasta que la introducción de las computadoras en la década del ’70 sentó las bases definitivas de lo que hoy conocemos como tecnología del control numérico computarizado (CNC).

En pocas palabras, el control numérico computarizado es el uso de una computadora para controlar y monitorear los movimientos de una máquina herramienta.

Uso CNC

El proceso que se muestra a continuacion comienza con la copia de cada una de las partes de la silla las cuales se separan y se posicionan en el espacio de trabajo.




\

Paso 2:

Después de colocar todas las piezas en la plancha, se se hace contour 2D de los contornos inferiores de encaje de las piezas, para generar el orificio que permite encajar las piezas. Estas “orejas” permiten ensamblar las piezas y tienen esa forma dado que el corte es interno y que la broca es redonda, no se pueden genrar angulos de 90° y esa es la razon de la forma de ese espacio. A diferencia de un mecanizado exterior que permite mecanizar este tipo de ángulos.




\

Paso 3:

Hay que generar el sacado para que las piezas encajen, utilizamos 2D pocket.




\

Paso 4:

Utilizando el 2D contour se hace el contorno de la pieza. Es importante incorporar los Tabs dado que estos sujetan la pieza en el proceso de corte. Los podemos ver en la segunda imagen como unos pequeños cuadrados en la linea azul y en detalle en la ultima imagen.

Paso 5:

Se genera el espacio de trabajo para el asiento al igual que para las piezas anteriores.




\

Paso 6:

Hacemos 2D contour para el contorno del asiento (lineas azules) y Trazo para sacar material de la pieza y generar la flexibilidad de esta. Estas ultimas son las lineas amarillas, las que entran en la pieza, llegando como a la mitad del espesor.

Modelo 3D Respaldo






\

Paso 7:

Para comenzar el respaldo, se debe generar el espacio de trabajo, en este caso un solido que cubre completamente la pieza 3D del respaldo.






\

Paso 8:

Luego se debe proyectar el respaldo en ese solido como se muestra a continuación, en ambos lados.






\

Paso 9:

Creamos un Setup (y desde ahora en adelante todos los pasos son dentro de él) y elegimos diferentes acciones para el mecanizado del respado. La primera acción es Adaptive Clearing para la parte concava de la pieza. En esta se extrae la mayor parte del material.






\

Paso 10:

La segunda acción es Parallel para la misma cara que la accion anterior, su función es pulir el sacado anterior. En esta ocación se debe cambiar la fresa, por una de punta redonda la cual genera un mejor acabado.






\

Paso 11:

Se duplica el paso anterior, pero el Parallel se gira en eun angulo de 90°, con el fin de pulir aun mas la parte concava del respaldo. Mantenemos la fresa del paso anterior.






\

Paso 12:

Los siguientes 3 pasos son iguales a los 3 pasos anteriores pero por la parte de atras del respaldo, por la parte convexa. Trabajando dentro del mismo Setup. La primera acción es Adaptive Clearing






\

Paso 13:

La segunda acción es Parallel para la misma cara que la accion anterior.






\

Paso 14:

Se duplica el paso anterior, pero el Parallel se gira en eun angulo de 90°.
