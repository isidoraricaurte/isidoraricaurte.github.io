Ambición de Diseño

Sistematizar un proceso de experimentación para llevarlo a un diseño distribuido

Ser parte desde una experimentación individual de forma doméstica, para poder entender los parámetros que efectivamente sirven para observar el desarrollo y evolución del proyecto. En primera instancia, esta experimentación consistirá en realizar un almacigo con separaciones en forma de grilla para poder medir los efectos de los diferentes tipos de combinaciones de plantaciones. Es necesario destacar que la grillas tendrán características independientes entre sí, debido a que la idea es testear al mismo tiempo distintas condiciones de sembrado. Estas condiciones son:

  • Primera Etapa:

    1. Factores Químicos
    2. Factores Físicos
    3. Factores Biológicos

  • Segunda Etapa:

    1. Asociación de plantas
    2. Formas de plantar: previa germinación o plantación directa
    3. Distancia de sembrado
    4. Tipo de riego

  • Tercera Etapa (Sistematización):

    1. Agua
    2. Suelo
    3. Ambiente

Como mencionamos anteriormente, este proceso se realizará por etapas en donde para poder pasar a la siguiente es necesario obtener los resultados esperados en la primera. De lo contrario, se deberán modificar algunos parámetros para poder superar la etapa en desarrollo.

Validación Sistematizada

Raspberry PI

A través de un sistema computacional que permite la grabación continua de 24 horas podremos realizar un monitoreo completo del crecimiento de las plantas. Además, utilizando este mismo aparato podremos sistematizar el riego de las mismas.

Con la recopilación de esta información se puede usar como referente en la replicación del experimento, sumandose en la base de datos conjunta generando una red colaborativa de información.
